# Construction Stock Management System 🏗️

A robust and efficient stock management solution tailored for the construction industry. This system provides a streamlined way to track materials, manage inventory across multiple categories, and generate professional reports.

**Developed with ❤️ by MUHAMMAD DANISH**

---

## 🚀 Overview

This application is designed to solve the complexities of material tracking in construction projects. Whether you're managing a godown or tracking specific components like Columns and Double TTs, this system provides the tools you need in a clean, user-friendly interface.

## ✨ Key Features

- **Inventory Tracking:** Real-time monitoring of 6 core construction categories (Godown, Double TT, Slop, Column, Phhty, and others).
- **Secure Access:** Built-in authentication system with secure password hashing to protect your data.
- **Reporting:** One-click generation of printable reports for individual entries or entire categories.
- **Auto-Serialization:** Systematic entry management with automatic serial number generation.
- **Responsive Interface:** Works seamlessly on desktops, tablets, and smartphones.

## 🛠️ Technology Stack

- **Backend:** Python / Flask
- **Frontend:** HTML5, CSS3, JavaScript (Bootstrap 5 & Tailwind CSS)
- **Database:** SQLAlchemy ORM with SQLite (PostgreSQL compatible)
- **Icons:** Bootstrap Icons

## 📥 Getting Started

### Prerequisites
- Python 3.11+
- pip (Python package manager)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/danishmalik7570-a11y/Construction-dashbaord-Invnetry-system.git
   cd Construction-dashbaord-Invnetry-system
   ```

2. **Set up a virtual environment:**
   ```bash
   python -m venv venv
   # Windows:
   venv\Scripts\activate
   # Linux/macOS:
   source venv/bin/activate
   ```

3. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Initialize & Run:**
   ```bash
   python main.py
   ```
   *The database will be automatically created on the first run.*
